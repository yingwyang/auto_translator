# 自动翻译插件 (Auto Translator)

自动将 AstrBot 发送的消息翻译成指定语言。

## 功能特性

- 🤖 **自动翻译**: 机器人发送消息时自动翻译成目标语言
- 🌐 **多语言支持**: 支持中文、英文、日文、韩文等多种语言
- 📝 **原文显示**: 可选择是否同时显示原文
- ⚡ **多种翻译源**: 支持本地翻译、在线 API、LLM 等多种翻译方式
- ⚙️ **灵活配置**: 支持通过命令动态调整翻译设置

## 安装

1. 将插件文件夹放入 AstrBot 的插件目录
2. 安装依赖：`pip install -r requirements.txt`
3. 重启 AstrBot

## 配置说明

在 `data/config/astrbot_plugin_auto_translator.yaml` 中配置以下选项：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enable_auto_translate` | bool | `true` | 是否启用自动翻译 |
| `source_lang` | string | `"ja"` | 源语言代码 |
| `target_lang` | string | `"zh"` | 目标语言代码 |
| `show_original` | bool | `true` | 是否显示原文 |
| `use_llm_translate` | bool | `false` | 是否使用 LLM 翻译（质量更好但更慢） |

### 常用语言代码

| 代码 | 语言 |
|------|------|
| zh | 中文 |
| en | 英文 |
| ja | 日文 |
| ko | 韩文 |
| fr | 法文 |
| de | 德文 |
| es | 西班牙文 |
| ru | 俄文 |

## 翻译源说明

插件支持多种翻译源，按以下优先级尝试：

### 速度优先模式（默认）
1. **本地翻译** (argostranslate) - 无需网络，速度快
2. **Bing 翻译** - 微软翻译 API
3. **Google 翻译** - Google 翻译 API
4. **MyMemory 翻译** - 备用在线翻译
5. **LibreTranslate** - 备用在线翻译

### 质量优先模式
1. **LLM 翻译** - 使用配置的 LLM 进行翻译，质量最好但速度较慢
2. 其他翻译源同上

## 使用方法

### 命令列表

| 命令 | 说明 |
|------|------|
| `翻译设置 <源语言> <目标语言>` | 设置翻译的语言方向 |
| `开启翻译` | 启用自动翻译 |
| `关闭翻译` | 禁用自动翻译 |

### 示例

```
翻译设置 ja zh
```
将源语言设置为日文，目标语言设置为中文。

## 工作原理

1. 当机器人发送消息时，插件会自动检测消息内容
2. 根据配置的源语言和目标语言进行翻译
3. 翻译完成后，以独立消息的形式发送翻译结果

## 注意事项

1. **网络依赖**: 除本地翻译外，其他翻译源需要网络连接
2. **翻译质量**: 不同翻译源的翻译质量可能有所差异
3. **LLM 翻译**: 开启后翻译质量更好，但会增加响应时间

## 依赖

- aiohttp >= 3.8.0
- argostranslate >= 1.9.0（可选，用于本地翻译）
