# 自动翻译插件 (Auto Translator)

自动将 AstrBot 发送的消息翻译成指定语言。

## 功能特性

- 🤖 **自动翻译**: 机器人发送消息时自动翻译成目标语言
- 🌐 **多语言支持**: 支持中文、英文、日文、韩文等多种语言
- 📝 **原文显示**: 可选择是否同时显示原文
- ⚡ **智能翻译源**: 自动选择最佳翻译源（LLM > 在线 API > 本地翻译）
- 🎯 **LLM 翻译**: 优先使用 LLM 进行高质量翻译
- 💾 **本地翻译**: 支持离线翻译，可预下载语言包
- ⚙️ **灵活配置**: 支持通过命令动态调整翻译设置

## 安装

1. 将插件文件夹放入 AstrBot 的插件目录
2. 安装依赖：`pip install -r requirements.txt`
3. 重启 AstrBot

## 配置说明

在 `data/config/astrbot_plugin_auto_translator.yaml` 中配置以下选项：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enable_auto_translate` | bool | `true` | 是否启用自动翻译 |
| `source_lang` | string | `"ja"` | 源语言代码 |
| `target_lang` | string | `"zh"` | 目标语言代码 |
| `show_original` | bool | `true` | 是否显示原文 |

### 常用语言代码

| 代码 | 语言 |
|------|------|
| zh | 中文 |
| en | 英文 |
| ja | 日文 |
| ko | 韩文 |
| fr | 法文 |
| de | 德文 |
| es | 西班牙文 |
| ru | 俄文 |

## 翻译源优先级

插件按以下优先级自动选择翻译源：

1. **LLM 翻译** - 使用 AstrBot 配置的 LLM，质量最好
2. **MyMemory 翻译** - 在线 API，稳定可靠
3. **LibreTranslate** - 在线 API，备用
4. **本地翻译** - argostranslate，无需网络，支持离线

## 使用方法

### 命令列表

| 命令 | 说明 |
|------|------|
| `翻译设置 <源语言> <目标语言>` | 设置翻译的语言方向 |
| `开启翻译` | 启用自动翻译 |
| `关闭翻译` | 禁用自动翻译 |
| `下载语言包 <源语言> <目标语言>` | 预下载语言包，避免翻译时等待 |

### 示例

```
翻译设置 ja zh
```
将源语言设置为日文，目标语言设置为中文。

```
下载语言包 ja zh
```
预下载日文到中文的语言包（如果没有直接的语言包，会自动下载中转语言包 ja->en 和 en->zh）。

## 预下载语言包

为了避免翻译时的等待时间，建议在使用本地翻译前预下载语言包：

```
下载语言包 ja zh
```

下载完成后，本地翻译会立即响应，无需等待。

## 工作原理

1. 当机器人发送消息时，插件会自动检测消息内容
2. 按优先级尝试各翻译源：LLM → MyMemory → LibreTranslate → 本地翻译
3. 翻译完成后，以独立消息的形式发送翻译结果

## 注意事项

1. **LLM 翻译**: 需要配置 LLM provider，翻译质量最好
2. **在线翻译**: MyMemory 和 LibreTranslate 需要网络连接
3. **本地翻译**: 无需网络，但需要预下载语言包
4. **语言包下载**: 首次使用本地翻译时会自动下载，可能需要几分钟

## 依赖

- aiohttp >= 3.8.0
- argostranslate >= 1.9.0（可选，用于本地翻译）
